# Supreethraj_Task18

This project demonstrates basic DOM manipulation using vanilla JavaScript.

## Overview
- Click any colored box to change its background color.
- Enter a name in the input field and click "Greet Me!" to update the header; empty input shows an alert.

## Files
- `index.html` — main page that loads the styles and script.
- `style.css` — styles for layout and color boxes.
- `script.js` — contains `updateGreeting()` and `changeBoxColor()` functions for interaction.

## How to run
1. Open `task18/index.html` in a web browser.
2. Interact with the color boxes or enter your name and click the button.

## Behavior examples
- Clicking the `blue` box sets its background to blue and text color to white.
- Entering `Asha` and clicking `Greet Me!` changes the header to `Hello, Asha`.
- Leaving the input blank and clicking the button shows an alert prompting for a valid name.

## Notes
- The color boxes set their text color to white for `blue`, `red`, and `green`.
- This is a simple demo for learning event handlers and DOM updates.
